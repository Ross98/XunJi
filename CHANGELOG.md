# Changelog

## v1.0 (2026-07-10)

初始发布。基于训记 Open API 的个人健身数据仪表盘。

### 核心功能

- **仪表盘**：训练摘要、体重趋势、饮食热量统计，今日一览
- **训练记录**：日/周/月/年维度查看训练详情、动作组数、容量、心率、热量、时长
- **训练日历**：日历视图快速导航训练日期，强度热力图
- **训练分析**：训练频率热力图、动作雷达图、类型分布、时长/心率/热量趋势
- **饮食追踪**：三餐记录、热量/营养素统计，支持搜索、增删改、自定义食物
- **饮食分析**：日/周营养汇总、宏量营养素比例、历史趋势
- **身体数据**：体重/体脂双轴图、BMI 计算、阶段变化统计
- **训练计划**：基于近期训练历史自动生成建议计划

### 技术架构

- Python 3.11+, FastAPI, uvicorn
- Jinja2 模板 + Chart.js 可视化
- SQLite 缓存层（避免重复 API 请求）
- httpx 异步 HTTP 客户端
- Docker 部署支持

### 配置要求

- 训记 App VIP 账号 + 4 组 Open API Key
- 参见 `.env.example` 配置说明
